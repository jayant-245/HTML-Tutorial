1. Understanding HTML and its Use Cases

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It describes the structure of a webpage using elements and tags.

Use Cases: Building website layouts, adding content like text, images, videos, forms, tables, etc.

2. Creating First HTML Page in VS Code

Open VS Code, create a new file with .html extension and paste this:

<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello World!</h1>
  </body>
</html>

Output: Displays "Hello World!" in large text.

3. HTML Structure

  • <!DOCTYPE html> - Defines the document as HTML5
  • <html> - Root element
  • <head> - Metadata, title, links
  • <title> - Page title shown in browser tab
  • <body> - Main content of the page
